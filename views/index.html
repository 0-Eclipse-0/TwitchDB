July 09, 2015


Delicious updates!
TwitchDB has recieved many updates including a new profile look. If you have not create an introduction or have a legacy introduction, feel free to make one! We now have a public database search. This search system is extremely basic, you can search by any field in the database so it will sometimes bring up some false positives. For example, if you search "LoL" for leauge, and someone has lol in their name it will bring up results for both. Eventually catagories may be added, and a complete revamp of the front page shall be done. Before that, the Feedback system needs to be re-integrated.

June 10, 2015


The Rewrite
I have redone TwitchDB from scratch to fix a vast array of bugs that everyone has been reporting. Thanks so much for all the feedback! Speaking of feedback, the streamer feedback system has TEMPORARILY been taken offline while I work out a much better system.
This rewrite is a very good thing, it will allow me to extend features much, much easier. the code has been simplified a great deal, and better design practices have been put in place.
